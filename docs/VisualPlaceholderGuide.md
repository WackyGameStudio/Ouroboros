# OUROBOROS: SWARM — 플레이스홀더 시각 규칙

> 목적: 정식 아트 전에도 역할과 위험을 색만으로 구분하지 않고 규칙을 검증한다.

| 대상 | 주 색 | 실루엣 | 내부 문양 |
| --- | --- | --- | --- |
| 머리 | Cyan | 원 | 양쪽 눈 |
| 실드 몸통 | Blue | 링 | 십자 축 |
| 공격 몸통 | Red | 삼각형 | 대각선 절개 |
| 레이저 몸통 | Magenta | 세로 직사각형 | 중앙 광선 |
| 제어 몸통 | Green | 마름모 | 십자 문양 |
| 추적체 | Orange | 마름모 | X 문양 |
| 투사체 | Yellow | 작은 원 | 밝은 코어 |
| 픽업 | Mint | 정사각형 | + 문양 |

- 역할은 색, 실루엣, 문양 중 최소 두 가지가 항상 다르다.
- 플레이스홀더는 `Assets/Ouroboros/Art/Placeholders/`에 32×32 Point Filter Sprite로 생성한다.
- 실제 아트 교체 시 Pivot은 중앙, PPU는 32를 기준으로 하고 판정 크기는 Sprite 경계와 분리한다.
